Tutorial
------
Recomended for python 3.6

Using git :

    $ git clone http://github.com/satriapasa/NewSatriaPy3


Python packages :

    $ pip3 install thrift
    $ pip3 install rsa
    $ pip3 install requests
    $ pip3 install linepy
    $ pip3 install gtts
    $ pip3 install googletrans
    $ pip3 install beauTifulSoup
    $ pip3 install html5lib

Run NewSatriaPy3:

    $ cd NewSatriaPy3
    $ python3 sb.py
